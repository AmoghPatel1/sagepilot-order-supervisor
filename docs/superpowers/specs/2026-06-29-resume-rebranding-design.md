# Design Spec: Resume Rebranding — AI Order Supervisor

**Date:** 2026-06-29  
**Status:** Approved

---

## Goal

Convert this project from a company-assessment artifact to a clean resume/portfolio project. Remove all assessment framing, rename from "Sagepilot Order Supervisor" to "AI Order Supervisor", and ensure every user-facing surface reflects a deliberately built personal project.

---

## Scope

Six files. No code logic changes. No new features.

---

## Changes

### `README.md`

- **Title**: `# AI Order Supervisor`
- **Opening**: Replace "A POC for a long-running AI supervisor..." with:
  - 2-sentence showcase intro explaining the problem and why it's interesting
  - Technical description paragraph follows (keeps existing substance)
- **Drop "POC"** everywhere it appears in body text
- **Project structure path**: `sagepilot-order-supervisor/` → `ai-order-supervisor/`

### `ARCHITECTURE.md`

- First line: `Sagepilot Order Supervisor is...` → `AI Order Supervisor is...`
- No other changes — content is already technically framed

### `DESIGN_NOTE.md`

Full rewrite. Keep all technical reasoning and trade-off tables. Change framing:

| Before | After |
|---|---|
| "The assignment asks for a system that..." | "The design challenge is building a system that..." |
| "The spec explicitly leaves orchestration open." | "Orchestration was an open choice." |
| "The spec mentions Sagepilot processes a high volume..." | "At production scale with high event volume..." |
| "For a 2-day POC, that setup cost would consume..." | "For this scope, that setup cost would..." |
| "For a 2-4 day POC, that complexity..." | "For this scope, that complexity..." |
| "being evaluated" | remove entirely |

Section heading "The Core Design Challenge" stays — reframe body paragraph to be about the technical tension (persistence + simplicity), not the assignment constraint.

### `frontend/app/page.tsx`

- `<h1>`: `Sagepilot Order Supervisor` → `AI Order Supervisor`

### `frontend/app/layout.tsx`

- `metadata.title`: `"Create Next App"` → `"AI Order Supervisor"`
- `metadata.description`: `"Generated by create next app"` → `"AI-powered e-commerce order lifecycle management"`

### `backend/app/main.py`

- `FastAPI(title=...)`: `"Sagepilot Order Supervisor"` → `"AI Order Supervisor"`

- README structural reorganization beyond what's stated
